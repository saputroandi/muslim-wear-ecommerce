# Muslim Wear Ecommerce

Express.js MVC starter for the Muslim Wear Ecommerce repository.

## Stack

- Express.js
- Handlebars views
- Static assets in `public/`

## Run locally

```bash
cp .env.example .env
npm install
npm run dev
```

## Routes

- `GET /`
- `GET /products`
- `GET /health`

## Scripts

- `npm run dev`
- `npm start`
- `npm run lint`
- `npm run format`
